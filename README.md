# 🏥 中药记忆大师

> 50味中药交互式记忆学习网页应用 —— 分类浏览、对比记忆、联想口诀、药对配伍、自我测验、错题本

## 📖 简介

**中药记忆大师** 是一款纯前端交互式中药学习工具，涵盖 50 味常用中药的功效分类、对比记忆、联想口诀、药对配伍等内容。适合中医药学生、从业者及爱好者利用碎片化时间进行高效记忆。

🌐 在线体验：https://wangchao0816.github.io/CNM/

## ✨ 功能特性

### 🌿 分类浏览
- 13 个中药分类（解表药、清热药、泻下药……）折叠浏览
- 按子类分组展示
- 药材详情页展示功效、记忆口诀、相似药材
- 支持药材名称/功效搜索

### 🃏 记忆卡片
- 四种学习模式：全部药材、按分类、按难度、错题本
- 3D 翻转卡片动画（正面：药材名 → 背面：功效+口诀）
- 收藏到错题本、标记已掌握

### 📊 对比记忆
- **23 组对比**，覆盖全部 50 味药材
- 同中求异，帮助辨析相似药材
- 每组含共同点、不同点、记忆口诀

### 💡 联想口诀
- 8 味重点药材的记忆口诀
- 关键词提取法辅助速记

### 🤝 药对配伍
- 6 组经典药对（麻黄+桂枝、大黄+芒硝……）
- 功效、代表方剂、配伍解析

### ✍️ 自我测验
- 四种模式：快速 5 题、标准 10 题、全部药材、错题本
- 两种题型：功效→药名、药名→分类
- 答题反馈 + 评分 + 错题重做

### 📕 错题本
- 收藏筛选、薄弱药材（错误≥3次）、近期复习
- 按错误次数排序
- 一键回顾所有错题

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| **Vue 3** (CDN global build) | 前端框架 |
| **Vue Router 4** (CDN) | 前端路由（Hash 模式） |
| **localStorage** | 用户数据持久化 |
| **CSS 自定义属性** | 主题与响应式设计 |

> 纯静态 SPA，无 Node.js / 构建工具 / 后端依赖，打开即用。

## 🗺️ 路由

## 📊 数据覆盖

- **药材**: 50 味（13 分类，含 HerbId 1-50）
- **对比组**: 23 组（涵盖全部 50 味药材）
- **口诀**: 8 味重点药材
- **药对**: 6 组经典配伍
- **题型**: 功效→药名 / 药名→分类

## 🚀 本地使用

直接用浏览器打开 `index.html` 即可运行，无需任何构建步骤。

```bash
# 或者通过本地服务器启动（推荐）
python -m http.server 8080
# 访问 http://localhost:8080
```
