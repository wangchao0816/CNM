### 2. 对比记忆组（关键相似药物对照）

**组1：麻黄 vs 桂枝（解表药双子星）**
- 共同点：均发汗解表，治风寒感冒
- 差异：麻黄"宣肺平喘、利水消肿"（偏实证）；桂枝"温通经脉、助阳化气"（偏虚证）
- 口诀：麻桂发汗解风寒，麻平喘水利肿专，桂温经脉助阳气，平冲降逆又化气

**组2：黄芩 vs 黄连 vs 黄柏（三黄清热燥湿）**
- 黄芩：清上焦肺火，兼止血安胎
- 黄连：清中焦胃火，泻火解毒力强
- 黄柏：清下焦湿热，退虚热
- 口诀：黄芩上焦肺火清，黄连中焦胃火灵，黄柏下焦湿热退，三黄燥湿解毒行

**组3：桑叶 vs 菊花（疏散风热明目对）**
- 共同点：疏散风热、平肝明目
- 差异：桑叶"清肺润燥"（偏肺）；菊花"清热解毒"（偏肝）
- 口诀：桑菊疏散风热清，桑叶润肺燥咳停，菊花解毒肝火平

**组4：柴胡 vs 葛根（升阳退热双杰）**
- 共同点：均能升阳、退热
- 差异：柴胡"疏肝解郁、升举阳气"（偏肝胆）；葛根"解肌退热、生津止渴、透疹"（偏脾胃）
- 口诀：柴胡疏肝解郁良，升举阳气少阳章；葛根解肌生津渴，透疹升阳止泻忙

**组5：牡丹皮 vs 赤芍（清热凉血活血对）**
- 共同点：清热凉血、活血化瘀
- 差异：牡丹皮"退虚热"（偏阴分）；赤芍"散瘀止痛"（偏止痛）
- 口诀：丹皮赤芍凉血活，丹皮退虚热骨蒸，赤芍散瘀止痛卓

**组6：川贝母 vs 浙贝母（二贝化痰）**
- 共同点：清热化痰、散结消痈
- 差异：川贝母"润肺止咳"（偏虚燥）；浙贝母"解毒散结"（偏实热）
- 口诀：川贝润肺燥咳宜，浙贝解毒散结奇，二贝清热化痰结，消痈散结两相依

**组7：牛膝 vs 川牛膝（二膝通经）**
- 共同点：逐瘀通经、利尿通淋
- 差异：牛膝"补肝肾、强筋骨、引血下行"（偏补益）；川牛膝"通利关节"（偏通利）
- 口诀：牛膝补肝肾强筋，引血下行功效真，川膝通利关节好，逐瘀通经两相亲

**组8：龟甲 vs 鳖甲（滋阴潜阳对）**
- 共同点：滋阴潜阳
- 差异：龟甲"益肾强骨、养血补心、固经止崩"（偏补益）；鳖甲"退热除蒸、软坚散结"（偏清退）
- 口诀：龟鳖滋阴潜阳同，龟甲强骨养心宫，鳖甲退蒸软坚结，固经止崩龟甲功

**组9：大黄 vs 芒硝（攻下双雄）**
- 共同点：泻下通便、清热
- 差异：大黄"攻积、凉血解毒、逐瘀通经、利湿退黄"（全面）；芒硝"润燥软坚、清热消肿"（偏软坚）
- 口诀：大黄攻积凉血毒，逐瘀通经退黄速，芒硝润燥软坚好，清热消肿两相助

**组10：附子 vs 肉桂（温里回阳对）**
- 共同点：补火助阳、散寒止痛
- 差异：附子"回阳救逆"（急救）；肉桂"引火归元"（引火下行）
- 口诀：附子回阳救逆急，肉桂引火归元宜，补火助阳散寒痛，温里回阳两相依

### 4. 药对/配伍记忆法

- **麻黄+桂枝**：发汗解表增强（麻黄汤）
- **大黄+芒硝**：攻下热结（大承气汤）
- **当归+川芎**：补血活血（四物汤基础）
- **龟甲+鳖甲**：滋阴潜阳（二甲复脉汤）
- **金银花+连翘**：清热解毒（银翘散）
- **附子+肉桂**：回阳救逆、补火助阳（肾气丸思路）

## 数据结构规范

### 中药数据模型
```typescript
interface Herb {
  id: number;                    // 1-50
  name: string;                  // 药名
  category: string;              // 功效大类（如：解表药）
  subCategory: string;           // 子类（如：发散风寒药）
  effects: string[];             // 功效列表
  memoryGroup?: string;          // 对比记忆组名（如："麻黄-桂枝组"）
  memoryTip: string;             // 记忆口诀/提示
  similarHerbs: number[];        // 相似药物ID列表
  difficulty: 'easy' | 'medium' | 'hard';  // 记忆难度
  image?: string;                // 药材图片URL（可选）
  isFavorite: boolean;           // 是否收藏进错题本
  mistakeCount: number;          // 错误次数
  lastReviewed: Date;            // 最后复习时间
}
